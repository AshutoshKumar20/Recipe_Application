# Recipe_Application
🍽️ KhudPaka — Recipe Finder App
A responsive recipe search application built with React + Vite, powered by the free MealDB API. Search thousands of recipes worldwide, view ingredients, and watch cooking tutorials directly from the app.

✨ Features
🔍 Search recipes by name in real time
⚡ Quick search chips — Biryani, Pasta, Chicken, Pizza, Sushi
🖼️ Recipe cards with image, category badge, cuisine area, and ingredients
▶️ Watch button — opens YouTube tutorial or source link
📱 Fully responsive — works on mobile, tablet, and desktop
🚫 Empty states — handles no results and pre-search gracefully

🛠️ Tech Stack

React 18            UI components and state management
Vite                Fast development server and build tool
JavaScript ES6+     Async/await, array methods, destructuring
CSS3                Custom styling, Flexbox, CSS Grid
MealDB API          Free recipe data source
Google Fonts        Playfair Display + DM Sans typography
